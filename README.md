## Código!

Para compilar los ejemplos basta con usar el compilador de c que tenemos instalado en linux:

```
    $ cc sum.c -o sum
    ```

Algunas pruebas para hacer con la consola (código de la clase 1):

```
$ ./count3 < prueba1.txt
6 15 18
$ ./sum < prueba2.txt
5
```
